# 🦖 Dinocraft

A 2D Minecraft-style browser game with a dinosaur theme — built with HTML5 Canvas and vanilla JavaScript.

**[▶ Play it live](https://matoskahol.github.io/dinocraft/)**

---

## About

Dinocraft is an original block-building survival game that runs entirely in the browser. Explore a procedurally generated world, mine ores, craft tools, discover fossils, and find glowing dino eggs hidden underground and in the treetops.

## Features

- Procedurally generated world with caves, biomes, and ore veins
- 4 biomes: Jungle, Plains, Desert, and Snow
- Mine coal, iron, gold, and diamonds
- Dino-themed secrets: fossils and glowing dino eggs
- Full day/night cycle with dynamic lighting
- Crafting system (planks, torches, chests)
- Inventory system
- Mobile-friendly with on-screen controls

## How to Play

| Action | Control |
|---|---|
| Move | A / D or Arrow keys |
| Jump | W or Space |
| Mine block | Left click |
| Place block | Right click |
| Hotbar slot | Keys 1–8 |
| Inventory | E |
| Crafting | C |

## Tech

- Vanilla JavaScript (no frameworks, no libraries)
- HTML5 Canvas 2D
- Runs in any modern browser

## Run Locally

Just open `game.html` in a browser. No build step, no dependencies.

```bash
git clone https://github.com/matoskahol/dinocraft
cd dinocraft
open game.html   # or just double-click it
```

## Deploy on GitHub Pages

1. Push this repository to GitHub
2. Go to **Settings → Pages**
3. Set source to **Deploy from a branch** → `main` → `/ (root)`
4. Your game will be live at `https://matoskahol.github.io/dinocraft/`
